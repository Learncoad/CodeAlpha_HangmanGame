# 🎮 Hangman Game (Python)

## 📌 Description

This is a simple Hangman Game developed in Python as part of the CodeAlpha Internship. The game allows users to guess letters to form a hidden word.

## 🚀 Features

* Random word selection
* User-friendly gameplay
* Limited attempts
* Input validation

## 🛠️ Technologies Used

* Python

## ▶️ How to Run

1. Clone the repository
2. Run:

   ```bash
   python game.py
   ```

## 📂 Project Structure

* main.py → Game logic
